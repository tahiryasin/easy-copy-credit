== Description ==
Automatically appends a customizable citation to copied text on your WordPress site.
== Installation ==
This section describes how to install the plugin and get it working.
1. Upload [`easy-copy-credit`](https://github.com/tahiryasin/easy-copy-credit/archive/refs/heads/master.zip) to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Update the configuration from Settings > Easy Copy Credit
